# Modelos de computación #Tema 

Un modelo de computación describe como se computa la salida de una función dada una entrada. Además de como se organizan las unidades de computación y las unidades de memoria. El uso de modelos permite calcular la complejidad computacional de un algoritmo. Podemos clasificar estos modelos en:
