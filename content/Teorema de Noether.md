# Teorema de Noether

Si las ecuaciones de Euler-Lagrange del movimiento, son invariantes para un cambio de coordenadas, entonces, existe una integral de movimiento. Quiere decir que, para cada simetria que se cumpla, una magnitud física se conserva.
